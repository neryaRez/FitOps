10. Migration Notes
============================================================

Do not remove Base44 immediately.

Recommended replacement order:
  1. Keep frontend working.
  2. Clean branding: title, favicon, README.
  3. Add backend health check.
  4. Replace userApi.js with API Gateway calls.
  5. Replace progressApi.js with API Gateway calls.
  6. Replace aiApi.js with OpenAI-backed Lambda flow.
  7. Replace Base44 auth with Cognito.
  8. Remove base44Client.js and @base44/sdk.
  9. Remove Base44 Vite plugin if no longer needed.

Security reminders:
  - Never put OpenAI API keys in the frontend.
  - Frontend must never access DynamoDB directly.
  - Use API Gateway + Lambda as the backend boundary.
  - Use Cognito JWT tokens for user identity.
  - Keep S3 photo bucket private.
  - Use least-privilege IAM.
